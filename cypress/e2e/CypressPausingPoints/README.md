# Overview

This folder holds a handful of pathways in which can be used to test quickly. This can help eliminate manually going through the application and have portions of it done autonomously.

## Getting Started

* Make sure to have "experimentalStudio": true in cypress.json
* Have Cypress 9.3.0 or higher

- Start by picking a pathway and making a copy of it. This copy is a starter template in which you can quickly make the test. Once you have done that, run the test on Cypress Studio and wait until it is finished to take control and make tests using the recording mode.
